# MULTI-CLASS-REAL-TIME-OBJECT-DETECTION-AND-TRACKING-USING-YOLOv8-WITH-EDGE-DEVICE-OPTIMIZATION
This project presents a real-time multi-class object detection and tracking system using YOLOv8 optimized for edge devices. Traditional deep learning models require high computational power and are not suitable for deployment on low-resource devices like Raspberry Pi and Jetson Nano.
